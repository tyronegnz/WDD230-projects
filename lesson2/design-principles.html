Design Principles

Contrast

Taco Bell

Taco Bell Site

This page does a great job when it comes to using contrast. The difference in colors make the page more atractive and alive. The yellow and purple complement each other.

Repetition

Apple

Apple Site

Apple's website uses the principle of repetition by using the same pattern in fonts and also by repeating shapes. Repetition is well used principle in marketing companies.

Alignment

Netflix

Netflix Site

This homepage uses alignment by justifying it's text and images. There is also alignment in the spacing which works to make the webpage look less clustered. It is very easy on the eyes.
